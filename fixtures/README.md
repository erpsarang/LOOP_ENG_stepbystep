# CSV fixtures

이 디렉터리의 파일은 자동 테스트와 품질 게이트에서 사용하는 고정 데이터입니다. 기계가 사용하는 경로와 기대값의 단일 기준은 `expectations.js`이며, 이 문서는 같은 내용을 사람이 읽기 쉽게 설명합니다.

## `valid.csv`

- 목적: 공백이 있는 정수, 소수, 음수처럼 지원되는 정상 `amount` 형식 검증
- 기대 합계: `1500.5`
- 기대 오류 행 수: `0`
- 기대 종료 코드: `0`

## `invalid-amount.csv`

- 목적: 숫자가 아닌 값과 빈 값을 제외하고 유효한 금액만 합산하는 동작 검증
- 기대 합계: `30.5`
- 기대 오류 행 수: `2`
- 기대 경고 수: `2`
- 기대 종료 코드: `0`

## `missing-amount-column.csv`

- 목적: 필수 `amount` 컬럼 누락 오류 검증
- 기대 결과: `오류: amount 컬럼을 찾을 수 없습니다.`
- 기대 종료 코드: `1`

fixture 내용이나 기대 결과를 변경할 때는 `expectations.js`와 이 문서를 함께 갱신해야 합니다. `test.js`와 `verify.js`는 공통 기대값 모듈을 직접 사용합니다.
